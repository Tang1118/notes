# JS 常用的循环遍历

[JS常用的循环遍历你会几种](https://juejin.cn/post/6966390357005172773)

js 常用循环：

## 数组遍历：

- for
- for in
- for of
- forEach
- some、every
- filter、map
- find、findIndex
- reduce、reduceRight

## 对象遍历

- for in：使用 for in 循环时，返回的是所有能够通过对象访问的、可枚举的属性，既包括存在于实例中的属性，也包括存在于原型中的实例。如果只需要获取对象的实例属性，可以使用 hasOwnProperty 进行过滤。
- Object.keys
- Object.values
- Object.entries
- Object.getOwnPropertyNames
