## React 支持 CSS Modules 联想

css 文件：
```css
.App {
  background-color: rgba(0, 0, 0, 0.08);
}

.app-header {
  background-color: white !important;
}

.AppLogo {
  height: 60px;
  width: 60px;
  float: left;
}
```

jsx 文件：

`设置 => 插件 => 搜索 React CSS Modules 下载使用`
