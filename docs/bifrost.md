# Bifrost 彩虹桥

彩虹桥（Bifrost，Bifröst，古北欧语：Bilröst）在北欧神话中是连结阿斯加德（Asgard）和 米德加尔特（中庭/Midgard）的巨大彩虹桥，意即“摇晃的天国道路”。
