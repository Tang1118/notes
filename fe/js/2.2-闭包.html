# 闭包

闭包是一个可以访问外部作用域的函数，即使这个外部作用域已经执行结束

# 外部作用域

内部函数可以访问外部函数中定义的变量（形参和实参），即使外部函数已经执行完成

(function (p) {
  
  let x = 1
  
  setTimeout(() => {
    
    console.log(x) // 1   实参
    
    console.log(p) // 20  形参
    
  }, 100)
  
})(10)
